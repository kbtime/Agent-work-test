# Skill 4：课程解析器

## Skill 名称

course_parser

## Skill 目标

解析讲师课纲文件，提取课程结构信息。

## 输入

课程文件目录：

```
02_课程大纲库/01_原始课纲/
```

支持文件类型：

pdf
pptx
docx
md

## 输出

```
02_课程大纲库/02_课纲解析结果/
```

## 使用 Prompt

```
课程解析Prompt.md
```

## 执行步骤

步骤1
扫描课程文件目录。

步骤2
读取课纲文本。

步骤3
调用 Prompt：

```
课程解析Prompt
```

步骤4
生成解析结果 Markdown。

## 输出示例

```
销售谈判技巧_解析结果.md
```

内容包括：

课程主题
适用对象
能力点
模块结构
教学方式
