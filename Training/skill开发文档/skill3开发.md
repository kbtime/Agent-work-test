# Skill 3：需求汇总器

## Skill 名称

demand_aggregator

## Skill 目标

将多个需求 JSON 合并成需求池，并筛选重点需求。

## 激活关键词

合并需求，汇总需求

## 输入

```
01_培训需求库/03_需求提取结果/*.json
```

## 输出

```
01_培训需求库/04_需求汇总池/
```

生成文件：

全部需求汇总.json
高优先级需求.json
待确认需求.json

## 执行步骤

步骤1
读取所有需求 JSON。

步骤2
合并需求列表。

步骤3
根据规则筛选高优先级需求。

筛选规则：

需求紧急程度 = 高
或
业务影响包含关键指标

步骤4
识别待确认需求。

筛选规则：

```
是否适合培训解决 = 部分适合
或
是否适合培训解决 = 待确认
```

步骤5
生成汇总 JSON。

## 输出 JSON 示例

```json
{
  "需求总数": 12,
  "高优先级需求数": 4,
  "待确认需求数": 2
}
```
