# Skill 2：需求处理器

## Skill 名称

demand_processor

## Skill 目标

将培训需求 Excel 文件转换为结构化需求 JSON。

激活关键词：提取需求，分析需求，转化需求，研究需求

## 输入

Excel 文件目录：

```
01_培训需求库/01_原始Excel/
```

Excel 文件包含字段：

需求标题
需求来源部门
涉及岗位
问题描述
培训目标
业务影响

## 输出

Markdown 文件：

```
01_培训需求库/02_MD转换结果/
```

JSON 文件：

```
01_培训需求库/03_需求提取结果/
```

## 使用 Prompt

```
Agent-work-test/Training/dmeo/需求提取prompt.md
需求提取Prompt.md
```

## 执行步骤

步骤1
扫描目录：

```
01_培训需求库/01_原始Excel/
```

步骤2
调用脚本：

```
excel_to_md.py
```

将 Excel 转换为 Markdown。

输出：

```
02_MD转换结果/*.md
```

步骤3
读取 Markdown 内容。

步骤4
调用 Prompt：

```
需求提取Prompt
```

提取结构化需求信息。

步骤5
生成 JSON。

## 输出 JSON 示例

```json
{
  "需求ID": "DEMAND-2026-001",
  "需求标题": "门店销售主动销售能力不足",
  "需求来源部门": "销售部",
  "涉及岗位": ["门店销售"],
  "问题现象": "员工不会主动介绍产品",
  "能力类型": ["销售能力","客户沟通"],
  "培训目标": "提升主动销售能力",
  "需求紧急程度": "中",
  "需求关键词": ["主动销售","成交引导"],
  "是否适合培训解决": "是"
}
```

## 返回结果

```json
{
  "processed_files": 5,
  "generated_demands": 12,
  "output_directory": "01_培训需求库/03_需求提取结果/"
}
```
