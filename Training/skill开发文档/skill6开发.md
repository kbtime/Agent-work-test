# Skill 6：需求课程匹配器

## Skill 名称

demand_course_matcher

## Skill 目标

根据培训需求，从课程能力库中匹配最合适课程。

## 输入

需求：

```
01_培训需求库/04_需求汇总池/高优先级需求.json
```

课程：

```
02_课程大纲库/03_课程能力汇总库/
```

## 使用 Prompt

```
需求课程匹配Prompt.md
```

## 输出

```
04_知识检索库/需求课程匹配结果/
```

文件：

匹配结果.json
匹配报告.md

## 执行步骤

步骤1
读取需求。

步骤2
加载课程能力库。

步骤3
根据能力类型、岗位、关键词筛选候选课程。

步骤4
调用 Prompt：

```
需求课程匹配Prompt
```

步骤5
生成匹配结果。

## 输出 JSON 示例

```json
{
  "需求ID": "DEMAND-2026-001",
  "推荐课程": [
    {
      "课程ID": "COURSE-001",
      "课程名称": "销售谈判技巧",
      "匹配分数": 0.91,
      "推荐讲师": ["张三"]
    }
  ]
}
```
