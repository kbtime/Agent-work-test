# Skill 5：课程能力构建器

## Skill 名称

course_builder

## Skill 目标

将课程解析结果转成标准课程能力对象。

## 输入

```
02_课程大纲库/02_课纲解析结果/
```

## 输出

```
02_课程大纲库/03_课程能力汇总库/
```

## 生成文件

课程结构卡片.md
课程摘要.md
能力标签.json
讲师版本映射.json

## 执行步骤

步骤1
读取课纲解析结果。

步骤2
生成课程结构卡片。

步骤3
生成课程摘要。

步骤4
生成能力标签 JSON。

步骤5
生成讲师映射。

步骤6
更新课程索引。

## 输出 JSON 示例

```json
{
  "课程ID": "COURSE-001",
  "课程名称": "销售谈判技巧",
  "能力类型": ["销售能力","客户沟通"],
  "核心能力点": ["需求挖掘","成交引导"],
  "适用岗位": ["门店销售"]
}
```
